# Society Maintenance Managing System - Standalone Version

A fully functional, portable web application for managing residential society operations with complete Local Storage integration.

## Features

- **No Backend Required**: Uses browser Local Storage for all data
- **Fully Portable**: Download and run anywhere without installation
- **Complete Functionality**: Admin portal, resident portal, billing, payments, notices, complaints
- **Premium Payment Gateway**: UPI, Card, Net Banking with receipt generation
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Quick Start

1. Download and extract all files to a folder
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)
3. The system is ready to use!

## Data Storage

All data is stored in browser Local Storage:
- Data persists across browser sessions
- Each browser stores data independently
- Clear browser data to reset the system

## System Requirements

- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)

- Minimum 1024x768 screen resolution recommended

## Offline Usage

The system works completely offline once loaded. CDN resources are cached by the browser.

## Data Backup

To backup your data:
1. Open browser DevTools (F12)
2. Go to Application/Storage tab
3. Export Local Storage data
4. Save as JSON file
